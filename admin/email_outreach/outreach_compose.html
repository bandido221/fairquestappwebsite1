📨 Compose Outreach Email

Step 1 — Who Are You Sending To?

Select which contact type(s) to email. For each fair, the system uses that type's email address. If a fair doesn't have it, it falls back to their default contact. Select multiple types to send a separate email to each.
⚠ None of the selected fairs have contacts set up yet. You can still enter emails manually in Step 3.

Step 2 — Write Your Email

Click a token to insert it — replaced with each fair's real data before sending:
{{fair_name}} {{city}} {{state}} {{contact_name}} {{website}}

Step 3 — Confirm Recipients

You can edit any field before sending.
