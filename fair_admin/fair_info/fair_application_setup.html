Manage Application Forms

How application forms work: You can create one form for each applicant type — Artisan/Craft Vendors, Food Vendors, and Performers. Each form has its own questions and contract. Once a form is marked active, vendors can apply through it. Click Edit Application to add your own questions or paste your vendor contract.
